# 🍎 mac-Os

A macOS-inspired UI built using React and Vite.

This project aims to recreate the macOS desktop experience in the browser — including dock, windows, animations, and app interactions.

---

## 🚀 Tech Stack

- React
- Vite
- JavaScript
- CSS (or Tailwind if you add later)

---

## 📦 Installation

Clone the repository:

```bash
git clone git@AnkitPaul7:AnkitPaul7/mac-Os.git
cd mac-Os
npm install
npm run dev
```
